Detailed Description

Class used to do all Transformation related functions. This class is based on the Singleton design pattern. The Transformation class holds:

  • a static instance of itself (Singleton).
  • the height of the window.
  • the width of the window.
  • the ranges of the game logic (default [-4,4]x[-3,3]).
