Detailed Description

This is an abstract class which represents a Vehicle. This class is derived from

See also
Entity. A vehicle has:
  • a driving speed.
  • a movement speed (for horizontal and vertical movement in the grid).
  • a bool to check if the vehicle has crashed or not.
